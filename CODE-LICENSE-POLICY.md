# Code License Policy

This repository uses a split-license model so text and runnable code are unambiguous:

- **Book text and instructional prose:** Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International (CC BY-NC-SA 4.0).
- **Code samples and project code:** MIT License (see [`LICENSE-CODE`](LICENSE-CODE)).

## What counts as "code"

The MIT license applies to source code files and executable examples in:

- `/code/` (if present)
- `/free/content/examples/`
- `/free/project-solutions/`
- `/paid/content/examples/`
- `/paid/project-solutions/`

The MIT license does **not** automatically apply to chapter text, explanatory prose, or other non-code documentation.

## Contributor attribution policy for non-original code

If you add code that you did not author:

1. Confirm the upstream license permits inclusion and redistribution.
2. Keep the original copyright and license headers when required.
3. Add a short source attribution comment at the top of the file (or in the nearest README when file headers are not practical).
4. Include any required third-party notices in this repository.
